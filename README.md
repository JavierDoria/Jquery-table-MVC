# Aplicación Web con ASP.NET MVC

## Descripción
Desarrollo de una aplicación web utilizando ASP.NET Core MVC,Integración de jQuery y el plugin DataTables para la visualización dinámica de datos, 
incorporando funcionalidades de búsqueda, paginación, ordenamiento y exportación a Excel, PDF e impresión.

## Tecnologías utilizadas
- ASP.NET Core MVC
- C#
- SQL Server
- Razor
- -jQuery/DataTables
- HTML5, CSS3
- Bootstrap
